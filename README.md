# AI-Based Billboard Size Estimation Using Computer Vision

## Overview

This project presents an AI-powered computer vision solution for estimating billboard dimensions from a single mobile photograph. The proposed system automatically detects billboards, estimates their width and height, calculates the surface area, and provides a confidence score for each prediction.

The project was developed as a commercial proof-of-concept to demonstrate how Artificial Intelligence can automate outdoor advertisement measurement and support infrastructure assessment.

---

## Problem Statement

Traditional billboard measurement requires manual field surveys, making the process time-consuming, labour-intensive, and prone to human error. This project explores how computer vision can automate billboard measurement using only a smartphone image.

---

## Objectives

- Detect billboards from a single mobile image.
- Estimate billboard width, height, and surface area.
- Generate confidence-based predictions.
- Evaluate the feasibility of AI-assisted outdoor infrastructure measurement.

---

## Methodology

The proposed workflow consists of:

Image Capture
→ Billboard Detection
→ Perspective Analysis
→ Dimension Estimation
→ Area Calculation
→ Confidence Score
→ Result Display

## Results

The prototype demonstrated encouraging performance during real-world testing.

### Sample Output

- Width: **9.76 ft**
- Height: **9.70 ft**
- Area: **94.67 sq ft**
- Confidence Score: **88%**
